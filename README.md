# KIZUKU 大宮 公式サイト

出張リラクゼーションマッサージ店「KIZUKU 大宮」の静的ウェブサイトです。系列店「[出張マッサージ EN八王子](http://enhachiouji.com/)」のトーンを踏襲しつつ、出張専門サービスの魅力が伝わるように再構成しています。

## プロジェクト構成

```
.
├── index.html          # トップページ
├── menu.html           # メニューページ
├── staff.html          # スタッフ紹介
├── policy.html         # 施術に関する注意事項
├── contact.html        # お問い合わせ
├── css/
│   ├── style.css       # 共通スタイル・テーマカラー
│   └── responsive.css  # ブレークポイントごとの調整
├── js/
│   ├── main.js         # ナビゲーション開閉・共通インタラクション
│   └── form-validation.js # フォームバリデーションと送信処理
└── images/             # 画像プレースホルダーを格納予定
```

すべてプレーンな HTML/CSS/JavaScript のみで構成されており、追加のビルド工程は不要です。ローカルで確認する場合は任意の HTTP サーバーを立てるか、ブラウザで各 HTML ファイルを開いてください。

## フォーム送信設定（Formspree）

1. [Formspree](https://formspree.io/) にログインし、新しいフォームを作成します。送信先メールアドレスに `omiya.kizuki@gmail.com` を指定してください。
2. ダッシュボードで発行されるフォーム ID（`https://formspree.io/f/xxxxx` の形式）をコピーし、`contact.html` の `form` 要素 `action` 属性を差し替えます。
3. reCAPTCHA やスパム対策のオプションを有効にする場合は Formspree の説明に従い追加設定を行ってください。
4. テスト送信を行い、フォーム送信完了メッセージが表示されること、メールが受信できることを確認します。

> メール送信処理は JavaScript の `fetch` API を利用しています。通信エラーや Formspree からのエラーレスポンス時には日本語のエラーメッセージを表示するようになっています。

## 画像の差し替え

- 現在は `https://i.pravatar.cc/300?img=xx` などのプレースホルダー画像を使用しています。本番運用時には `images/` ディレクトリに適切な画像を配置し、各 HTML 内の `src` を差し替えてください。
- ヒーローセクションの背景画像は Unsplash の写真を使用しています。ブランドの世界観に合わせて差し替える場合は、`css/style.css` 内の `.hero` セレクタで指定しています。

## カラーパレット・フォント

- プライマリカラー：`#2c6b66`
- セカンダリ（背景）カラー：`#f2ede4` / `#f7f5f0`
- アクセントカラー：`#b3833b`
- フォント：`Noto Sans JP`（本文・見出し）、`Noto Serif JP`（装飾的要素）

Google Fonts を `<link>` タグで読み込んでいるため、追加設定は不要です。自社フォントに切り替える場合は `index.html` などに埋め込んでいるフォント読み込みを変更し、`css/style.css` のフォントファミリーを更新してください。

## 料金テキストについて

メニューの料金・説明文は系列店「出張マッサージ EN八王子」の公開情報を参考に作成しています。価格の正式決定後は `menu.html` と `index.html`（メニュー概要）を更新してください。
